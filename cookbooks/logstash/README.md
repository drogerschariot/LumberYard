logstash Cookbook
=================
Installs Logstash, Elasticsearch 0.20.6, and/or Kibana 3.


Requirements
------------
- Java
- Nginx/Apache if installing logstash::kibana


Usage
-----
#### logstash
Install Logstash.

#### logstash::elasticsearch
Install Elasticsearch 0.20.6

#### logstash::kibana
Install Kibana 3
